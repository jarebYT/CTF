Bienvenue sur mon CTF

Premièrement, c'est quoi un CTF ?

Un CTF (ou CAPTURE THE FLAG) est un jeu où le but est de résoudre des énigmes pour trouver des "flags" (chaînes de caractère ou autre), un peu dans le style d'un escape game.

Voici les règles du jeu :

  • Règle 1: NE PAS FAIRE DE F12 pour accéder au code source (sinon ça n'a plus aucun intérêt de jouer, donc svp).

  • Règle 2: Vous avez le droit d'utiliser des outils externes (sites internet, logiciels, IA, etc..).

  • Règle 3: Ne partagez pas les solutions.
